Application Name: Text-Based Adventure Game
Description: This is a Python-based text adventure game.

How to Run Locally:
1. Clone the repository.
2. Install Python 3.10 or higher.
4. Run the application:
   python textgame.py

How to Run with Docker:
1. Pull the Docker image from Docker Hub:
   docker pull dhal0108/textbasedgame
2. Run the container:
   docker run -p 8000:8000 dhal0108/textbasedgame
